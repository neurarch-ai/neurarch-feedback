## 🧠 Neurarch 是什么？

Neurarch 把 ML 模型开发变成可视化工作流。你不用再手写 PyTorch 把每一层接起来 —— 在画布上拼搭架构，让 AI agent 帮你扩展，最后一键导出可直接运行的代码。

### 你可以做什么

- 🎨 **可视化模型设计** —— 在无限画布上拖拽、连线、配置 layer。形状推断实时进行，建模时就能发现维度不匹配。
- 🤖 **AI agent** —— 用自然语言描述你想要的（比如 "给 CIFAR-10 写一个小型 ResNet"），直接得到一个可以编辑的架构。
- 📦 **一键代码导出** —— 从任何画布生成干净的 PyTorch 代码、部署包、可嵌入 widget 或 Colab notebook。
- 🚀 **云端 GPU 训练** —— 通过 Modal 启动训练，不用碰任何基础设施。免费登录用户用模拟训练；Pro 用真 GPU。
- 📚 **自动生成文档** —— 把你的模型变成 model card、学习手册、PPT 或部署 README，并按你的目标读者定制。
- 🧩 **模板和模型库** —— 从已知架构（Transformer、U-Net、ViT …）开始，而不是从空白画布开始。
- 🔑 **BYOK 或托管模式** —— 用你自己的 Claude / Gemini / Groq key（只存在浏览器里），或用 Pro 的托管代理。

完整功能列表、定价和在线 demo 请访问 **[neurarch.com](https://neurarch.com)**。

## 提交前请注意

- **先搜一下现有 issue。** 可能已经有人报过了。
- **一个 issue 只讲一件事。** 不要把多个 bug 塞到同一张工单。
- **写具体一点。** 浏览器 + 操作系统、复现步骤、期望行为 vs 实际行为。截图和 console 报错非常有用。
- **不接收一般性 ML 求助。** 这里只跟 Neurarch 这个产品相关，不负责帮你 debug 自己的模型。
